# LinkedIn Coffee Chats Feature

A new feature for the LinkedIn mobile app that helps users schedule coffee chats based on availability, interests, and profile matching.

## Features

- View potential coffee chat matches based on:
  - Career interests
  - Personal interests
  - Availability
  - Location
  - Connection degree
- Filter and search functionality
- Schedule chats or send messages directly
- Modern LinkedIn-style UI/UX

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the Metro bundler:
```bash
npm start
```

3. Run on Android:
```bash
npm run android
```

Or run on iOS:
```bash
npm run ios
```

## Tech Stack

- React Native
- TypeScript
- React Navigation
- React Native Vector Icons
- React Native Safe Area Context
